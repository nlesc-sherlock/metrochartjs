Optional colors

colors: object[]
  • Defined in options.ts:20

The colors defined by colors specify the colors with which to draw metrolines. Each color must be an Object literal with properties name (a descriptive name of the color, such as 'dark blue' or 'seagreen') and hexcode (the hexadecimal representation of a color in #RRGGBB).

If no colors are specified, the default metroline colors from MetroChart.defaultOptions.colors are used. In any case, if there are more unique metrolines than colors, modulo math is used to wrap the colors.

  • name: string
